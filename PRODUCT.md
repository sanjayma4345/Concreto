# Product

## Register

brand

## Users

Designers, product managers, and engineers using AI coding tools who demand production-grade design output. They know AI-generated UIs have quality problems. They want precise control, not vague guidance.

## Product Purpose

Concreto provides a complete design vocabulary for AI coding agents: 80+ deterministic rules, ML-enhanced detection, and 35+ commands for every design operation. 

Success metrics:
1. User can steer AI with precision (specific commands, not vague requests)
2. Output passes professional design review
3. Zero "AI made this" tells in final product

## Brand Personality

Architect, engineer, craftsman. Concreto speaks with technical precision: structural truth, material honesty, functional beauty.

**Three-word personality: industrial, precise, honest.**

## Anti-references

The antithesis of:
- Generic AI tool marketing (purple gradients, neon, glassmorphism)
- SaaS clichés (hero metrics, identical card grids, sparklines)
- Hedging ("maybe", "consider", "perhaps")
- Educational framing (solve, don't teach)
- Decorative excess

## Design Principles

1. **Practice what you preach** - Concreto must pass its own audits
2. **Show, don't tell** - The output IS the demo
3. **Industrial precision** - Every element engineered for purpose
4. **Material honesty** - No decoration without function
5. **Structural clarity** - Clear hierarchy, logical flow

## Rule Categories

| Code | Category | Description |
|------|----------|-------------|
| A | Accessibility | WCAG compliance, keyboard, screen reader |

| Level | Priority | Action |
|-------|----------|--------|
| CRITICAL | Blocker | Fix immediately |
| HIGH | Major | Fix before ship |
| MEDIUM | Quality | Should fix |
| LOW | Polish | Nice to have |

## Accessibility

WCAG 2.1 AA minimum:
- Contrast ratios verified programmatically
- Keyboard navigation for all interactions
- `prefers-reduced-motion` respected
- Semantic HTML first, ARIA supplementary
- Readable copy (8th-grade level baseline)
